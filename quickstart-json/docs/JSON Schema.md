## JSON Schema介绍

JSON Schema is a vocabulary that allows you to annotate and validate JSON documents.

JSON Schema 是一个词汇表，允许您注释和验证 JSON 文档。

JSON Schema 是用于验证 JSON 数据结构的强大工具。然而，通过阅读它的规范来学习使用它就像通过查看它的蓝图来学习驾驶汽车。如果您只想拿起杂货，则无需知道电动机是如何组合在一起的。因此，本书旨在成为 JSON Schema 的友好驾驶教练。它适用于那些想要编写并理解它，但可能对构建自己的汽车不感兴趣的人——呃，编写他们自己的 JSON 模式验证器——只是现在。
