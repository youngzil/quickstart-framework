liteFlow能最大程度上解耦，支持即时调整策略的一个中间件



在项目内有2个简单的测试工程示例：
liteflow-test-spring
liteflow-test-springboot
分别对应了spring和springboot环境下的示例工程，可以运行Runner进行启动(会连带启动测试用例)
